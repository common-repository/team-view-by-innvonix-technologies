A fully responsive Team Members plugin to provide Team view in different layouts also as slider or grid view, with some effects such as overlay effect. New material colors, easy adding shortcodes and many more. Other options will be provided very soon in new update.


== Description ==
Team Member Plugin Slider and Grid view,Overlay Effects with Mobile and Tablet Responsive using simple Shortcodes. 

Innvonix Team Member is a WordPress Plugin with Slider and Grid Layouts,Overlay Effects with easy setup with shortcodes. Modern design and Faster loading with quick response Light weight, Touch Supported, WordPress Plugin.We will come soon with new layouts, customization options and features.

A few notes about Innvonix Team Slider:
 
*   Shortcodes [team-slider] & [team-grid] & [team-grid-overlay] & [team-slider-overlay] with "limit", "view" "order", "column", "content_limit", "group" attributes.
*   Fully customizable Slider and Grid Layouts with Overlay Effects.
*   Attributes will work like limit = post limit, view = layouts like *Square *Round *Rounded, Order = ASC /  DESC, Column = Columns and slider Items per page, Content_limit = text limit on view.  
*   Team member, Team member slider, slider, quote, team member slider, comments, team member groups.
*   Groups as like post categories. you can filter using groups attribute. 
*   Requires at least WordPress 4.4 version
*   Tested Upto WordPress 4.4 to WordPress 4.8 


== Installation ==
Installation Steps are as follow.

1. Upload `team-view-by-innvonix.zip` to the `/wp-content/plugins/` directory
2. Unzip the archive `team-view-by-innvonix.zip`.
3. Activate the plugin through the \'Plugins\' menu in WordPress

Or

1. Go through Plugins > Add New > Search Team Member and Install 
2. If you have already downloaded the plugin go through Plugins > Add New > Upload and select the team-member.zip after installation activate plugin.
